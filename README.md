# Курсовое сервис приложение рассылок

## Описание:

Этот проект представляет собой инструмент для работы с рассылками

## Использование:

1. Клонируйте репозиторий:
```
https://github.com/Seriynaya/coursework4.git
```


2. Установление зависимостей:
```
pip install django
```


3. Подключение к базе данных:
```
[postgres]
host = localhost
port = 5432
user = postgres
password = 5432
```


4. Запуск проекта:
```
python main.py
```


## Структура проекта:

1. mail/ - приложение для работы с сервис рассылкой.
2. users/ - приложение для работы с пользователями.
